# Technical Summary: pi-free-providers

## Project Overview
Pi extension providing 4 free model providers for Pi coding
 agent:
- **Kilo** - 300+ models via OAuth (free + paid)
- **OpenCode Zen** - Free models, no signup
- **OpenRouter** - 29+ free models (needs free API key)
- **NVIDIA NIM** - 70B+ models (needs free API key)
